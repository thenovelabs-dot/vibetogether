# 앱인토스 심사 전 체크리스트

최종 업데이트: 2026-06-17

## 현재 범위

앱인토스 1차 빌드는 게시판과 프로덕트 중심 커뮤니티로 제출한다.

- 포함: 홈, 게시판, 게시글/댓글, 프로덕트, 프로덕트 댓글, 검색, 공지, 마이페이지, 문의, 신고, 이용약관, 개인정보 처리방침
- 제외: 모임, 모임 신청/관리, 커피챗, 위치 기반 만남 탐색
- 보류: 토스 로그인, 앱인토스 SDK 연동, 콘솔 배포

## 완료

- `VITE_APP_TARGET=apps-in-toss` 타깃 분리
- `npm run build:ait` 추가
- 앱인토스 타깃에서 `/meetup*` 라우트 비활성화
- 앱인토스 타깃에서 모임 탭 제거
- 앱인토스 타깃에서 홈 피드의 모임 데이터 요청 제거
- 앱인토스 타깃에서 마이페이지/유저 프로필의 모임·커피챗 진입점 제거
- 앱인토스 타깃에서 검색 결과의 모임 조회 제거
- 앱인토스 타깃에서 Google 로그인 버튼 비활성화
- 앱인토스 타깃에서 프로덕트 외부 링크 CTA 및 SNS 링크 숨김
- 게시글 신고 동선 있음
- 프로덕트 신고 동선 있음
- 댓글/대댓글 신고 동선 있음
- 문의하기 동선 있음
- 이용약관/개인정보 처리방침 라우트 있음
- 마이페이지 탈퇴 동선 있음

## 출시 전 필수

- 토스 로그인 연동
- 토스 사용자 식별자와 Supabase `users.id` 매핑 방식 확정
- 앱인토스 콘솔 앱 등록
- 앱인토스 배포 스크립트/API 키 설정
- 앱인토스 샌드박스 또는 실기기 WebView QA
- 앱인토스 내비게이션 바와 자체 상단바/하단탭 충돌 확인
- 신고 접수 후 운영자 처리 절차 문서화
- 금칙어/부적절 콘텐츠 운영 기준 정리

## 수동 QA

- `/meetup`, `/meetup/new`, `/meetup/:id`, `/meetup/:id/applications` 직접 접근 시 홈으로 이동한다.
- 모바일 하단 탭에 모임 탭이 없다.
- 홈 피드에 모임 카드가 없다.
- 검색 placeholder가 게시글/프로덕트 기준으로 보인다.
- 검색 결과에 모임 섹션이 없다.
- 로그인 화면에 Google 로그인 버튼이 노출되지 않는다.
- 프로덕트 상세에 외부 서비스 바로가기/SNS 링크가 노출되지 않는다.
- 게시글 더보기에서 신고하기가 열린다.
- 프로덕트 더보기에서 신고하기가 열린다.
- 댓글/대댓글 신고 버튼이 열린다.
- 약관/개인정보 처리방침에서 모임, 커피챗, Google OAuth 문구가 앱인토스 빌드에 노출되지 않는다.
- 문의하기 모달이 정상 전송 또는 실패 토스트를 보여준다.

## 정책 리스크 메모

- 외부 링크는 앱인토스 정책상 제한적으로만 허용된다. 앱인토스 1차 빌드에서는 프로덕트 외부 링크를 숨긴다.
- 토스 로그인 전까지는 로그인 기반 기능을 출시 가능 상태로 보지 않는다.
- UGC 운영 리스크 때문에 신고/문의 동선은 심사 전 반드시 실제로 접근 가능해야 한다.
- 모임/커피챗은 만남성 서비스로 해석될 수 있으므로 앱인토스 1차 빌드에서는 유지하지 않는다.
